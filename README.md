Q1) What is the difference between id and class in CSS and HTML?
Answer: In Html, the main difference between id and class is , id is used to give a specific(unique) name to one element whereas the class is used to group multiple elements together.
In CSS, the main difference between id and class in Css is that in id it is used to target a unique element using selector (#) and in class , it is used to target multiple groups of elements using the (.) selector.
